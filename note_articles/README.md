# note記事生成パイプライン

## 概要

建築基準法改正ガイド（6章構成）を、note.com向けの個別記事に変換したコンテンツ群。
各記事は独立して読めるが、全体でガイドへの導線を形成する。

## ソースコンテンツ

- ガイド本体: `AI-Architecture-Circle/public/guide/index.html`
- 6章構成の建築基準法改正（2025年4月施行）完全対応ガイド

## 記事フォーマット

各記事は以下の構成で統一:

1. **タイトル** - SEOを意識した【】付きタイトル
2. **フック** - 読者の課題意識に刺さる導入文
3. **本文** - ガイドの正確な事実情報をベースにした解説
4. **CTA** - 無料ガイドダウンロードへの誘導
5. **著者ブロック** - senaのプロフィールリンク

## 事実チェックリスト（必ず遵守）

- 施行日: **2025年4月**
- 分類: **構造を問わない**（木造に限らない）
- 新2号: 2階建て以上 または 延べ面積200m2超
- 新3号: 平屋かつ200m2以下
- 仕様規定（壁量計算等）外注費: **5〜15万円/棟**
- 省エネ計算外注費: **5〜15万円/棟**
- 合計: **10〜30万円/棟**
- 許容応力度計算（300m2超等）: 15〜50万円/棟
- 捏造した事例・統計は一切使用しない

## 投稿パイプライン（別途構築）

このリポジトリにはnote APIへの投稿コードは含まない。
記事の投稿は以下のいずれかで行う:

1. **手動投稿** - 各mdファイルの内容をnoteエディタにコピペ
2. **API投稿スクリプト** - 別途構築予定（note API利用）

## トーン・ボイス

- senaの一人称視点（カジュアルな日本語）
- 専門用語は使うが、必ず平易な説明を添える
- 断定的ではなく「〜かもしれません」「〜のケースが多いです」のような語調
- 読者は建築実務者（設計事務所の建築士）を想定
