# IT導入補助金（デジタル化・AI導入補助金 2026）対応アクションプラン

## 概要

Archi-Prisma Design Worksの各ソフトウェアをIT導入補助金対象ツールとして登録し、
顧客が補助金を使って導入できるようにする。

## 対象製品と補助金適格性

| 製品 | IT導入補助金 | ものづくり補助金 | 小規模持続化 |
|------|:---:|:---:|:---:|
| KOKOME（省エネ計算AI） | **対象** | 対象（顧客申請） | 対象（顧客申請） |
| KOUZOU（構造計算AI） | **対象** | 対象（顧客申請） | 対象（顧客申請） |
| SpotPDF（図面差分チェック） | **対象** | 対象（顧客申請） | 対象（顧客申請） |
| KAKOME（現場写真指示） | **対象** | 対象（顧客申請） | 対象（顧客申請） |
| Compass（PM） | **対象** | 対象（顧客申請） | 対象（顧客申請） |
| AI建築サークル | **非対象** | 非対象 | 非対象 |

## IT導入補助金の補助額

| 枠 | 補助額 | 補助率 |
|----|--------|--------|
| 通常枠（1-3プロセス） | 5万〜150万円 | 1/2 |
| 通常枠（4+プロセス） | 150万〜450万円 | 1/2 |

**SaaSサブスクリプション: 最大2年分が補助対象**

## 即座にやること（3月30日まで）

### 1. gBizID Prime取得
- URL: https://gbiz-id.go.jp/
- マイナンバーカードがあれば15分で取得可能
- 法人代表者（櫻本聖成）が申請

### 2. SECURITY ACTION宣言（★★二つ星）
- URL: https://www.ipa.go.jp/security/security-action/it-hojo.html
- 2-3営業日で宣言番号発行
- 加点対象になるので★★二つ星を推奨

### 3. 必要書類の準備
- [ ] 履歴事項全部証明書（3ヶ月以内発行）
- [ ] 法人税の納税証明書（直近事業年度）
- [ ] 各製品の機能説明資料（スクリーンキャプチャ、機能一覧）
- [ ] 価格表（税込・税抜明記、プラン別定価）
- [ ] 導入事例（実際の利用顧客の実績）

### 4. 販売実績の確認
**重要**: 登録するツールには販売実績が必要。
各製品に最低1名の有料ユーザーがいることを確認。

## 登録フェーズ（3月30日〜）

### 5. IT導入支援事業者登録
- 2026年3月30日(月) 10:00 受付開始
- URL: https://it-shien.smrj.go.jp/itvendor/flow/
- 仮登録 → 本登録 → 審査

### 6. ITツール登録（同時進行）
- リード製品: KOKOME または Compass
- 1製品ずつ順次登録

## 重要な注意点

1. **交付決定前の購入は対象外** → 顧客に申請→承認→購入の流れを案内
2. **ECサイト利用は不可** → セルフサービスだけでなく、ベンダーの関与が必要
3. **価格表は固定** → 「〜円から」のような表記はNG
4. **AI建築サークルは対象外** → コミュニティ/教育サブスクはITツールに該当しない

## 顧客向けマーケティング

LP・各製品ページに以下を追加:
- 「IT導入補助金対象ツール（申請中）」バッジ
- 「最大1/2の補助金で導入可能」の訴求
- 補助金活用の申請サポートの案内
